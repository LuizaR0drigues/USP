* Foi a primeira rede neural descrita algoritmicamente, sendo criado pro Frank Rosenblatt, um psicólogo, em 1958.
* É utilizado para classificar padrões do tipo **Linearmente separáveis**
* Consiste em **um único neurônio** com pesos sinápticos ajustáveis e bias
* Se os dados de treino pertencerem a classes linearmente separáveis, o algoritmo converge, posicionando um hiperplano entre as duas classes.

Os pesos sinápticos são denotados por $W_1, W_2, ..., W_m$
As entradas são denotadas por $x_1,x_2,...,x_m$
O bias é denotado por $b$
Sendo a função de ativação denotado por:
$$v = \sum_{i=1}^{m} W_i x_i + b$$

O objetivo do algoritmo é classificar corretamente um conjunto de exemplos, $x_1,x_2,...,x_m$, em um de uma classes $C_1$ ou $C_2$
Sendo classificado como $C_1$ se a saída $y = +1$ e como $C_2$ se $y = -1$. Estas regiões são separadas pelo hiperplano dado por:
$$ \sum_{i=1}^{m} W_i x_i + b = 0$$
Ele tua como uma fronteira de decisão para problemas de classificação com duas dimensões e duas classes.
### Teorema de Convergência
O ajustes dos pesos sinápticos ocorrem forma interativa utilizando o algoritmo de convergência do Perceptron.

O bias $b(n)$ é tratado coo um peso associado a uma entrada $+1$
Vetor de entrada: $x(n) = [+1, x_1(n), x_2(n),...,x_m(n)]^T$
Vetor de pesos: $W(n) = [b, W_1(n), W_2(n),...,W_m(n)]^T$
$$ v(n) = \sum_{i=0}^{m} W_i x_i + b = W^T(n)x(n)$$
* $W^T(n)x(n) = 0$ define um hiperplano de separação
* $W^T(n)x(n) > 0$ para todo vetor $x \in C_1$ 
* $W^T(n)x(n) <0$ para todo vetor $x \in C_2$ 

Se o n-ésimo vetor $x(n)$ é corretamente classificado pelo vetor $w(n)$ na n-ésima iteração do algoritmo,nenhuma correção é feita no vetor de pesos
* ${W(n+1) = W(n)}$, se $W^Tx(n) > 0$  e $x(n) \in C_1$
* ${W(n+1) = W(n)}$, se $W^Tx(n) <= 0$  e $x(n) \in C_2$
Caso contrário o vetor será atualizado:
* ${W(n+1) = W(n)}+N(n)x(n)$, se $W^Tx(n) <= 0$  e $x(n) \in C_1$
* ${W(n+1) = W(n)}-N(n)x(n)$, se $W^Tx(n) > 0$  e $x(n) \in C_1$
Sendo $N(n)$, a taxa de aprendizado que controle o ajuste dos pesos

A saída do neurônio é computada utilizando a função sinal $sgn(.)$, dada por:
$$sgn(v) = \left[ \begin{array}{ll} +1, & \text{se } v>0 \\ -1, & \text{se } v<0 \end{array} \right.$$
ou de forma compacta:
$$y(n) = sgn[W^T(n)x(n)]$$

Foi utilizada também a resposta desejada $d(n)$ para cada exemplo:
$$d(n) = \left[ \begin{array}{ll} +1, & \text{se } x(n) \in C_1\\ -1, & \text{se } x(n) \in C_2 \end{array} \right.$$Resultando em:
$$ W(n+1) =  W(n) + N[d(n) - y(n)]x(n)$$
Onde $d(n) - y(n)$ é o sinal de erro
Os pesos são corrigidos de acordo com o valor do produto interno $W^T(n)x(n)$

### Gradiente Descendente
$$E² = (d(n) - y(n))² =  (d(n) - W^T(n)x(n))²$$
$$W_i(n+1) = W_i(n)-N \frac{d}{dW_i} E²$$
$$\frac{d}{dW_i} E² =  \frac{d(d(n) - y(n))²}{dW_i} = 2(d(n) - W^T(n)x(n)) - x_i$$
Como 2 é uma constante pequena, simplificamos para:
$$\frac{d}{dW_i} E² =  (d(n) - W^T(n)x(n)) - x_i$$
